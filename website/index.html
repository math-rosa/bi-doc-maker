Por que usar o BI Doc Maker?

Diferenciais técnicos projetados para empresas que exigem excelência, rastreabilidade e segurança.

Local e Privado

Não utiliza IA externa nem APIs para analisar o PBIP. Metadados, regras de negócio e modelo semântico permanecem na sua máquina.

Documentação Automática

Mapeia tabelas, colunas, relacionamentos, medidas DAX, consultas Power Query, páginas e recursos do projeto PBIP.

Qualidade Executiva

Gere DOCX com visual premium, Markdown técnico e HTML imprimível. Inclui Mermaid, leitura DAX, regras Power Query e dicionário de dados.

Como Funciona

Um fluxo simples que respeita a estrutura oficial do Power BI.

1

Salve como PBIP

No Power BI Desktop, salve seu relatório no formato de projeto (.pbip) para expor a estrutura de metadados.

2

Gere a Documentação

Aponte o BI Doc Maker para a pasta do projeto, escolha os formatos e personalize título, logo e cores.

3

Arquivos Prontos

A documentação é salva em Doc_BI. O HTML pode ser aberto no navegador para salvar como PDF.
